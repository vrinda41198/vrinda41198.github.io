Software Development Intern (June 2018 - July 2018)

I worked on developing an Intelligent Document Processor - a data parser system which would use the internal Optical Character Recognition software to scan images of documents, and then classify the data to different user-defined constructs.

My main contribution was towards developing the algorithms that would help identify the required data based on the information provided by the user. This product is immensely useful for professions that require scanning large documents frequently- for instance, in law or in finance- where the user can describe the data they want to extract, and, the model will highlight the closest possible results.
